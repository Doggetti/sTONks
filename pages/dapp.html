Install MetaMask
  1. Download and install official MetaMask browser extension
  2. Create a new wallet, MetaMask password and Secret Recovery Phrase
  3. Add BSC Testnet chain - used for testing purposes only
Add BSC Testnet chain
  1. Open Networks menu in MetaMask
  2. Click Add Network
  3. Setup a network:
    Network Name: BSC Testnet
    New RPC URL: https://data-seed-prebsc-1-s1.binance.org:8545/
    Chain ID: 97
    Currency Symbol: BNB
    Block Explorer URL: https://testnet.bscscan.com
  4. Click Save
Add BNB to your wallet
  1. Copy your wallet's address in MetaMask
  2. Go to Binance Smart Chain Faucet page
  3. Paste your wallet's address
  4. Click Give me BNB
  5. Wait a few seconds for the transaction to be completed
Connect your wallet
  1. Go to Stonksapp website
  2. Click the wallet icon in the top right corner of the website
  3. Select your account in MetaMask
  4. Click Next
  5. Click Connect
  6. Click Sign
Swap
  1. Connect your wallet on Stonksapp website
  2. Type BNB amount or click Replace icon and type STONK amount
  3. Type correct slippage amount (15% - 19%)
  4. Click Swap
  5. Confirm transaction in MetaMask
Place Stop loss & Take profit orders
  1. Connect your wallet on Stonksapp website
  2. Click on the Order tab
  3. Choose Stop loss and Take profit
  4. Specify the number of tokens you want the order to be based on
  5. Select/type the difference or the exact price at which the order should be executed
  6. Set the slippage to 50 (only the required amount of gas will be used e.g. the limit will be 500000 and the transaction will only use 55000 which is 11% slippage)
  7. Give the contract access to the number of tokens entered above and then sign the future execution of the order via Metamask
  8. When the above conditions are met the order will be executed
